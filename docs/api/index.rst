API reference
=============

The public API is re-exported at the package top level; the most common entry
point is :func:`surface_renewal.run_surface_renewal` configured by
:class:`surface_renewal.PipelineConfig`.

   from surface_renewal import run_surface_renewal, PipelineConfig, Calibration
